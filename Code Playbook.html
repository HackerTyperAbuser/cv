Tables

In HTML tables allows us to arrange data into rows and columns. The table element allows us to output tabular data in a webpage. The table row (tr) contain table cells, either header cells (th) or data cell (td), with content insdie. A table can have a caption.
Example of tables:

Forms

An HTML form is used to collect user input. THe user input is most often sent to a server for processing. The form element is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons etc...







Forms can further be styled using CSS and JavaScript for its fullest functionality.
